# QENEX Brand Guidelines

## The mark
- Always render via `wordmark.js` or use canonical SVGs in `01-logo/svg`.
- Never recreate the geometry by hand.
- Never stretch, skew, recolor outside palette, add shadows, or rotate.

## Minimum sizes
- Wordmark: 96px wide on screen, 24mm wide in print.
- Below that, use the Q monogram alone.

## Clear space
Equal to ½ the cap-height of the Q on every side.

## Color usage
- Primary gradient = identity. Use it for the mark on neutral surfaces.
- Cosmos backgrounds use the full gradient as a wash; place the cosmos variant of the mark on it.
- Knockout (white) for photo/video; never print white-on-white.
- Mono (#0A0E27) for single-color reproduction (foil stamp, embroidery, fax).

## Voice & tone
- Confident, precise, scientific.
- Avoid filler. One thousand no's for every yes.
- Never use exclamation points in marketing copy.

## Type pairings
- H1: Space Grotesk 500, -3% letter spacing.
- Body: Inter 400, 1.6 leading.
- Metadata / system text: JetBrains Mono, 12% letter spacing, uppercase.

## Misuse
Never:
- Place the mark on busy imagery without a halo or container.
- Outline the wordmark.
- Add drop shadows or bevels.
- Substitute the cyan core with another accent.
- Use the gradient on body type.
